# 第三方插件


## Markdown All in One
1. 对于如下的 markdown 结构，
    ```md
    # 文件主标题
    ## 第一部分
    ## 第二部分 
    ```
2. 生成时 section numbering 时，它生成的是
    ```md
    1. 文件主标题
    2. 1 第一部分
    3. 2 第二部分
    ```
3. 这导致这个文件里所有的段落都在 `1` 这个段落号之下。
4. 可以在设置项 `toc.levels` 中，把默认的 `1..6` 改为 `2..6` 来实现生成
    ```md
    1. 第一部分
    2. 第二部分
    ```
