# 括号匹配检测


## 实现
```js
function toBinary (n) {
    if ( n === 0 ) {
        return '0';
    }

    let stack = [];

    while ( n > 0 ) {
        stack.push(n % 2);
        n = Math.floor( n / 2 );
    }

    let str = '';
    while ( stack.length > 0 ) {
        str += stack.pop();
    }

    return str;
}
```
