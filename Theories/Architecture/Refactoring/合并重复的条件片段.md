## 例子
1. 假如我们有一个分页函数`paging`，该函数接收一个参数`currPage`，`currPage`表示即将跳转的页码。在跳转之前，为防止`currPage`传入过小或者过大的数字，我们要手动对它的值进行修正，详见如下伪代码：
    ```js
    var paging = function( currPage ){
        if ( currPage <= 0 ){
            currPage = 0;
            jump( currPage );    // 跳转
        }
        else if ( currPage >= totalPage ){
            currPage = totalPage;
            jump( currPage );    // 跳转
        }
        else{
            jump( currPage );    // 跳转
        }
    };
    ```
    改成：
    ```js
    var paging = function( currPage ){
        if ( currPage <= 0 ){
            currPage = 0;
        }
        else if ( currPage >= totalPage ){
            currPage = totalPage;
        }
        jump( currPage );    // 把jump函数独立出来
    };
    ```
